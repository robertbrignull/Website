Cloyster pattern

The idea for Cloyster is that it's made of concentric spheres and part spheres. So we have a black sphere, surrounded by most of a grey sphere, surrounded by two clam shell shapes. With spikes on of course.

What you'll need
  • 3.25mm knitting needles
  • Black, grey, white and purple double knitting wool
Knit the black body

With black wool, cast on 12 stitches
Row 1: k11 (1 before end), w&t
Row 2: p10 (1 before end), w&t
Row 3: k9 (2 before end), w&t
Row 4: p8 (2 before end), w&t
Row 5: k6 (4 before end), w&t
Row 6: p4 (4 before end), w&t
Row 7: k8 (to end)
Row 8: p11 (1 before end), w&t
Row 9: k10 (1 before end), w&t
Row 10: p9 (2 before end), w&t
Row 11: k8 (2 before end), w&t
Row 12: p6 (4 before end), w&t
Row 13: k4 (4 before end), w&t
Row 14: p8 (to end)
Repeat rows 1-14 twice more

Sew up and stuff the black body

Sew together the two cast on and bound off ends. On the top or bottom, insert needle through loops and pull together. Through the remaining hole stuff the sphere to your liking. Now pull together the other end in the same way.

Don't worry about neatness as all this sewing will be hidden.

Knit the inner grey shell

With grey wool, cast on 18 stitches
Row 1: k17 (1 before end), w&t
Row 2: p16 (1 before end), w&t
Row 3: k14 (3 before end), w&t
Row 4: p12 (3 before end), w&t
Row 5: k9 (6 before end), w&t
Row 6: p6 (6 before end), w&t
Row 7:k12 (to end)
Row 8: p17 (1 before end), w&t
Row 9: k16 (1 before end), w&t
Row 10: p14 (3 before end), w&t
Row 11: k12 (3 before end), w&t
Row 12: p9 (6 before end), w&t
Row 13: k6 (6 before end), w&t
Row 14:p12 (to end)
Repeat rows 1-14 once more

Knit the grey spike

With grey wool, cast on 5 stitches
Row 1: k5
Row 2: p5
Row 3: k2tog, k3 (4 st)
Row 4: p4
Row 5: k2tog, k2 (3 st)
Row 6: p3
Row 7: k2tog, k1 (2 st)
Row 8: p2
Row 9:bind off

Sew on the inner grey shell and spike

On the top or bottom, insert needle through loops and pull together. Place the black body inside the grey shell, hiding the stitching on the body. Sew the spike onto the top of the body, between the body and the shell.

The body should be slightly smaller than the shell so that you can pinch the shell together to make in crinkled. Sparsely sew the shell to the body in this crinkled way. Make sure to pinch together the top and bottom to make them come away from the body, then do maybe two or three pinches on each side. The idea is of course to make it look like Cloyster's crinkled shell.

Knit the outer purple shell (x2)

With purple wool, cast on 24 stitches
Row 1: k24
Row 2: p24

Row 3: k10, k2tog, k11 (1 before end), w&t (23 st)
Row 4: p21 (1 before end), w&t
Row 5: k8, k2tog, k9 (3 before end), w&t (22 st)
Row 6: p16 (3 before end), w&t
Row 7: k2, k2tog, k3 (6 before end), w&t (21 st)
Row 8: p9 (6 before end), w&t
Row 9: k1, k2tog, k2 (10 before end), w&t (20 st)
Row 10: p10 (to end)

Row 11: k5, k2tog, k5, k2tog, k5 (1 before end), w&t (18 st)
Row 12: p16 (1 before end), w&t
Row 13: k6, k2tog, k6 (3 before end), w&t (17 st)
Row 14: p11 (3 before end), w&t
Row 15: k3, k2tog, k3 (6 before end), w&t (16 st)
Row 16: p4 (6 before end), w&t
Row 17: k10 (to end), w&t
Row 18: p16

Row 19: k4, k2tog, k3, k2tog, k4 (1 before end), w&t (14 st)
Row 20: p12 (1 before end), w&t
Row 21: k4, k2tog, k5 (2 before end), w&t (13 st)
Row 22: p9 (2 before end), w&t
Row 23: k2, k2tog, k3 (4 before end), w&t (12 st)
Row 24: p4 (4 before end), w&t
Row 25: k8 (to end), w&t
Row 26: p12

Row 27: k2, inc, k6, inc, k2 (14 st)
Row 28: p14
Row 29: k2, inc, k8, inc, k2 (16 st)
Row 30: p18
Row 31: k18
Row 32: p18
Row 33: bind off

Sew on the outer purple shell

Sew together the two tails of the shells. Wrap the outer shell around what you've already made and sew the front corners of the outer shell onto the grey inner shell. Place the corners such that the outer shell is pushed away from the inner shell a bit. You can also sew the back half of the outer shell onto the inner shell if you want.

Knit the large purple spikes (x2)

With purple wool, cast on 5 stitches
Row 1: k5
Row 2: p5
Row 3: k2tog, k3 (4 st)
Row 4: p4
Row 5: k2tog, k2 (3 st)
Row 6: p3
Row 7: k2tog, k1 (2 st)
Row 8: p2
Row 9:bind off

Knit the small purple spikes (x4)

With purple wool, cast on 4 stitches
Row 1: k2tog, k2 (3 st)
Row 2: p4
Row 3: k2tog, k1 (2 st)
Row 4: p2
Row 5:bind off

Sew on all the purple spikes

Sew all of the spikes to the outer shell, place the two larger spikes at the top front of each half and the smaller ones at the bottom front and back.

Make the mouth and eyes

With white wool, create some stitches that look like two eyes and a mouth. With the expression, try to make it mischevious rather than happy or angry.
